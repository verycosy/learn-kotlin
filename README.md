# 접근제한자
## 패키지 스코프
- public(기본값) : 어떤 패키지에서도
- internal : 같은 모듈 내에서만
- private : 같은 파일 내에서만

## 클래스 스코프
- public : 클래스 외부에서
- private : 클래스 내부에서
- protected : 클래스 자신과 상속받은 클래스에서
